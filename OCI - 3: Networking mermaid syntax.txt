B. OCI VCN Routing and Security Mind Map
This mind map breaks down the key concepts of routing traffic and securing a VCN.

mindmap
  root((VCN Routing & Security))
    ::icon(fa fa-network-wired)
    Routing
      :Route Tables
      ::Set of rules for traffic LEAVING the VCN
      ::Implicit Local Routing (Inside VCN is automatic)
      ::Explicit Routing (Uses Longest Prefix Match)
      :Peering & Transit
      ::Local Peering (LPG)
      :::2 VCNs, same region
      ::Remote Peering (DRG)
      :::2 VCNs, different regions
      ::Transit Routing (DRG)
      :::Modern hub for many VCNs
    Security (Virtual Firewalls)
      :Security Lists (SL)
      ::Scope: Subnet
      ::Granularity: Broad
      ::Source/Dest: CIDR Block
      :Network Security Groups (NSG)
      ::Scope: VNIC (Instance)
      ::Granularity: Fine-grained
      ::Source/Dest: CIDR or another NSG
      ::Modern, application-centric approach
    Load Balancing
      :HTTP LB (Layer 7)
      ::Intelligent, content-based routing
      ::SSL Termination
      :Network LB (Layer 4)
      ::Max performance, ultra-low latency
      ::For non-HTTP traffic
